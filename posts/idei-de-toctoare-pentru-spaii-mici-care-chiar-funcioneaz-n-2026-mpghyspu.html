FAQ - Întrebări Frecvente

De ce este lemnul de fag mai bun decât cel de bambus pentru un set ustensile bucatarie?

Lemnul de fag este un lemn de esență tare, cu o structură mai omogenă și mai fină decât bambusul. Bambusul poate fi adesea prea dur pentru cuțite și conține mulți adezivi chimici folosiți în procesul de fabricație a plăcilor.

Cum pot elimina mirosurile neplăcute dintr-un tocător de lemn?

Pentru a împrospăta un set ustensile bucatarie din lemn, poți freca suprafața cu jumătate de lămâie și sare grunjoasă. Aceasta va dezinfecta și va extrage mirosurile de ceapă sau usturoi impregnate în fibre.

Este grosimea de 20mm prea mare pentru un spațiu mic?

Absolut nu. Grosimea adaugă stabilitate fără a mări amprenta la sol a produsului. Un tocător stabil este mult mai eficient într-un spațiu de lucru restrâns deoarece nu se va mișca în timpul utilizării.

Pot tăia carne crudă pe aceste tocătoare din lemn de fag?

Da, puteți. Totuși, conform recomandărilor FDA, este indicat să folosiți unul dintre tocătoarele din set exclusiv pentru carne și pe celălalt pentru legume/pâine, pentru a evita contaminarea.

Cât de des trebuie uleiat un set ustensile bucatarie din lemn?

Dacă îl folosiți zilnic, o dată la 4-6 săptămâni este ideal. Veți observa că lemnul are nevoie de ulei atunci când începe să arate „uscat” sau când apa nu mai rămâne la suprafață sub formă de bobițe.

Este lemnul de fag rezistent la mucegai?

Fagul are o rezistență naturală bună, dar niciun lemn nu este imun dacă este lăsat în apă. Uscarea verticală după fiecare spălare este esențială pentru a preveni acumularea de umiditate la bază.
